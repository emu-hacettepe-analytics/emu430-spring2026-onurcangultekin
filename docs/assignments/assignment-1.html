(a)

The video I choose is Veri Bilimi ve Endüstri Mühendisliği Üzerine Sohbetler - Mustafa Baydoğan & Erdi Daşdemir

Summary: Mr. Mustafa explains the main concepts related to data science and forecasting processes with real life applications. The talk focuses on data types, prediction models, the parameters affecting predictions and the consequences of these predictions in decision making processes. In addition, he also mentions ranking algorithms and the extent to which human contribution is still needed in the data and data prediction industry.

# ── 6. NA değerlerini doldurma ────────────────────────────────────────────────
new_data <- polls_us_election_2016  # orijinali bozmuyoruz

for (col in names(new_data)) {
  if (is.numeric(new_data[[col]])) {
    # Sayısal sütunlar: my_birth_year + k ile doldur
    new_data[[col]][is.na(new_data[[col]])] <- my_birth_year + k
  } else if (is.character(new_data[[col]]) || is.factor(new_data[[col]])) {
    # Karakter/faktör sütunlar: "ADINIZ_k" ile doldur
    fill_val <- paste0(my_first, "_", k)
    if (is.factor(new_data[[col]])) {
      levels(new_data[[col]]) <- c(levels(new_data[[col]]), fill_val)
    }
    new_data[[col]][is.na(new_data[[col]])] <- fill_val
  }
}
